Case Study

SayurKu Dashboard

Year: 2024 Role: Frontend Developer

The Challenge

Managing inventory and orders for a growing vegetable delivery service was becoming chaotic with spreadsheets. The goal was to create a centralized dashboard that provides real-time insights into sales, stock levels, and user activity.

The Solution

I designed and developed a responsive admin panel that visualizes complex data into digestible charts. The interface prioritizes clarity, allowing admins to quickly assess business health at a glance.

  • Real-time Data Visualization: Used Chart.js to render interactive sales graphs.
  • Responsive Design: Fully functional on mobile devices for on-the-go management.
  • Modern UI: Clean aesthetics with a focus on readability and ease of navigation.

Tech Stack

React Chart.js Tailwind CSS Vite
